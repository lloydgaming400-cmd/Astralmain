*FINDABLE ITEMS*

Items can be found by meeting hidden activity thresholds. The bot never reveals thresholds, drop chances, or counts to anyone. If asked, the bot responds: Some things reveal themselves in time.

*General Rules for All Found Items:*
- Spam under 3 characters does not count toward any threshold
- Repeated identical messages only count once per minute
- Bot commands do not count toward any threshold
- Daily thresholds reset at midnight every day
- Overall thresholds never reset unless specifically stated
- Items go straight to inventory when found
- A user can only hold one of each item at a time. If they already hold that item the drop is skipped and the threshold resets
